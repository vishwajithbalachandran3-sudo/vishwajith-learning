# Instructional Designer Portfolio

A responsive, accessible static portfolio website showcasing concept learning projects without audio or video.

## Preview

Open `index.html` in a browser, or run a local static server from this directory.

## Before publishing

1. Replace the placeholder email address in `index.html`.
2. Add a LinkedIn link and a downloadable PDF resume.
3. Confirm the preferred spelling of the displayed name.
4. Replace or expand concept samples with real, non-confidential work where permitted.
5. Deploy the folder with GitHub Pages, Netlify, or Vercel.

The sample company situations are fictional and should not be treated as operational safety policy.
